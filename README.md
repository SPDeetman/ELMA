# ELMA (ELectricity MAterials)
repository for the model on material use in the electricity sector as used in the following publication in Resources, Conservation & Recycling:

"Projected material requirements for the global electricity infrastructure – generation, transmission and storage"
Deetman, de Boer, van Engelenburg, van der Voet & van Vuuren
https://doi.org/10.1016/j.resconrec.2020.105200

corresponding author: deetman@cml.leidenuniv.nl

The dynamic stock model used is based on the ODYM model by Stefan Pauliuk, Uni Freiburg, Germany. For the original code & latest updates, see: https://github.com/IndEcol/ODYM

The read_mym module used to read IMAGE model files is provided by the courtesy of Maarten van den Berg

In order to run the model please specify user-specific paths tagged in the code as <YOUR_DIR>
